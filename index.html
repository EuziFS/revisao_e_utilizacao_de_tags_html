O termo 'HTML'

Em tradução ao significado de HTML temos uma linguagem de marcação de hipertexto usada para rendeziar páginas na web. Sendo reconhecido como uma linguagem padrão para comunicação entre páginas, os navegadores adotaram a sua interpretação e temos o que chamamos hoje de websites. No entanto, o HTML não se trata de uma linguagem de programação propriamente dita, o termo linguagem, indica que é uma forma padrão de marcação textual.

HTML é uma abreviação de Hypertext Markup Language - Linguagem de Marcação de Hypertexto. Resumindo em uma frase: o HTML é uma linguagem para publicação de conteúdo (texto, imagem, vídeo, áudio e etc) na Web. (W3C.br)

Estrutura do HTML

Sua forma é bem caracterísca: São textos em que, para realizar uma determinada ação nele, é colocado demarcadores com funções determinada a cada tipo de marcador. Parece confuso, porém é simples, os marcadores são chamados de TAGs e cada um faz uma ação específica no texto, podendo ser de negrito, itálico, marcação textual, sublinhado, taxado ou excluído, entre outros. Além disso ainda temos àqueles que são usados para criar listas, links para outros sites e diversas funções que vemos navegando na internet.

A combinação das diversas Tags existentes, a espertice do criador e a necessidade dos usuários fez, e ainda faz, surgir diversas páginas na internet, cheios de conteúdo e de novas técnicas, todos os dias.

Qual é o papel do HTML na internet

A internet vem crescendo constantemente. A cada dia novas interfaces e funções são criadas e também diversos proficionais se qualificam para trabalhar. É certo que a área está com com muita dificuldade de achar mão de obra qualificada para a quantidade de vagas disponíveis, mas ainda é concorrido se desejar entrar no ramo.

Áqueles que são qualificados são chamados desenvolvedores, ou programadores. São esses quem criam e recriam a internet como conhecemos. Porém, por trás de toda interface amigável, tem um trabalho excepcionalmente árduo destes profissionais.

Para entender melhor: a internet como a vemos e utilizamos parece simples, porém ela é subdividida entre o que a gente vê (chamado de client side ou lado do cliente) e o que a gente não vê, porém sabemos que existe, pois toda vez que realizados uma ação, ela nos responde (chamados de server side ou lado do servidor). Já percebeu que toda vez que realizamos algo na internet, na verdade estamos "conversando" com alguma coisa do outro lado? daí a gente pede algo e esse outro lado somplesmente nos responde. Essa é a ideia 'a grosso modo' que temos da internet.

O "lado do servidor" é uma máquina certo?! estas utilizam de impulsos de energia para saber o que fazer e também o que não fazer, porém o "lado do cliente" somos nós, seres vivos e ativos que quer entender tudo que faz. Nesse caso, impulsos elétricos não serão uma boa forma de comunicação. Nesse caso temos a utilização de interface gráfica, textos, cores, imágens, etc. E ai que encontramos o real intuito da utilização do HTML: nos proporcionar uma forma mais interessante de se comunicar com o meio tecnológico.
